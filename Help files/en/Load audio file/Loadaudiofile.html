Load audio file

This module allows to load an audio file for analysis.

Interface

  1. Preview window: Allows to see the signal that will be generated by the module.
  2. “Open file" button: Allows to select an audio file to load.
  3. Playback buttons: They allow you to start or stop the audio file playback. If a fragment is selected, only this part will be played.
  4. Fragment selection panel: Allows to select a fragment of the loaded audio. To select the start and end, you will have to write in the enabled boxes the time in which each tip will be placed, or by clicking on the display window.
  5. Information panel: Displays information about the duration of the file and its sample rate.
  6. “Load" button: Opens a new visualization window in which the audio file, or the selected fragment, is loaded.
